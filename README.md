# Data Structures in TypeScript
My implementations of data structures in TypeScript.  
Source files are inside the ts/ folder, while test files are inside
the test/ folder.  
<br/>
To compile, use
```console
$ npm run tsc
```

To run tests, use 
```console
$ npm run test
```

To generate documentation, run
```console
$ npm run typedoc
```

### Implemented Data Structures
* AVL Tree
* B-Tree
* Binary Heap
* Binary Search Tree
* Bit Set
* Hash Table
* Linked List
* Quick Find
* Quick Union
* Skip List
* Splay Tree
